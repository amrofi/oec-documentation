OEC Documentation

Section 2 Data Sources

All of the product data shown on the OEC site is classified using either SITC (Standard International Trade Classification) or HS (Harmonized System).

For historical SITC classification data (1962 - 2000), the OEC is using data from [1]. For more recent data (2001 - 2016), the OEC is using data provided by UN COMTRADE.

References

[1] R. Feenstra, R. Lipsey, H. Deng, and H. Ma Alysonand Mo, “World trade flows: 1962-2000,” National Bureau of Economic Research, 2005.
